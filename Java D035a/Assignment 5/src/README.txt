README

1) To begin program, open src folder as java project
2) navigate to Driver.java class file
3) edit second argument on Line 43 "statLab2.print(students,4);" from '4' to '0' for header only,
																		  '1' for low score only,
																		  '2' for high score only,
																		  '3' for average only,
																		  or keep as '4' for all
4) Click Run Driver.java
